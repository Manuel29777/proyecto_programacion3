# Proyecto Programación 3

Aplicación desarrollada con Flutter.

Funciones:

- Login de usuario
- Dashboard
- Consumo de API REST
- Lista enlazada (Linked List)
- Buscar usuarios
- Eliminar usuarios
- Conteo de ingresos
- Fecha de último acceso
- Cierre de sesión

Tecnologías usadas:

- Flutter
- Dart
- API JSONPlaceholder
- Estructuras de datos
